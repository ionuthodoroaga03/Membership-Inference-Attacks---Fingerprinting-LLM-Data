# EmoLit Dataset

This repository contains the **EmoLit dataset**, a multi-label fine-grained emotion dataset derived from literature.  

- **Dataset Version:** 1.0  
- **Original Authors:** Luis Rei & Dunja Mladenić  
- **License:** [CC BY-SA 4.0](http://creativecommons.org/licenses/by-sa/4.0/)  
- **Source / DOI:** [Zenodo](https://zenodo.org/record/7883954) (DOI: 10.5281/zenodo.7883954)  

For full attribution details, see [`ATTRIBUTION.md`](ATTRIBUTION.md).

---

## Description

- Sentences are taken from [Project Gutenberg](https://www.gutenberg.org/).  
- Contains **38 emotion labels** (+ neutral examples).  
- Semi-supervised dataset for emotion detection in literature.
